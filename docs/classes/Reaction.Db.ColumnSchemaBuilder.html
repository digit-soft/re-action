__construct()

__construct(string  $type, integer|string|array  $length = null, \Reaction\Db\DatabaseInterface  $db = null, array  $config = array())

Create a column schema builder instance giving the type and value precision.

The default implementation does two things:

  • Initializes the object with the given configuration $config.
  • Call [[init()]].

If this method is overridden in a child class, it is recommended that

  • the last parameter of the constructor is a configuration array, like $config here.
  • call the parent implementation at the end of the constructor.

Parameters

string $type

type of the column. See [[$type]].
integer|string|array $length

length or precision of the column. See [[$length]].
\Reaction\Db\DatabaseInterface $db

the current database connection. See [[$db]].
array $config

name-value pairs that will be used to initialize the object properties
